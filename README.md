# 🧠 Mental Health Risk Analysis (AI-Generated Dataset)

## 📊 Project Overview

This project analyzes mental health risk factors using an AI-generated dataset obtained from Kaggle. The dataset simulates real-world behavioral, demographic, and psychological variables to explore patterns associated with mental health outcomes.

The goal of this project is to identify key predictors of mental health risk and demonstrate strong exploratory data analysis (EDA) and data interpretation skills.

---

## 🎯 Objectives

* Explore relationships between lifestyle, demographic, and psychological variables
* Identify the strongest predictors of mental health risk
* Perform structured exploratory data analysis (EDA)
* Build visualizations that clearly communicate insights
* Develop a data-driven understanding of mental health factors

---

## 🗂️ Dataset Information

* Source: Kaggle (AI-generated dataset)
* Type: Simulated data representing mental health indicators
* Size: ~6,000+ rows, 20+ features

### Key Features:

* **Demographics:** age, gender, marital_status, education_level
* **Lifestyle:** sleep_hours, physical_activity, screen_time
* **Psychological Metrics:** stress_level, depression_score, mood_swings
* **Health Indicators:** panic_attack_history, substance_use
* **Target Variable:** mental_health_risk

> ⚠️ Note: This dataset is AI-generated and does not represent real individuals.

## 🔧 Project Workflow

### 1. Data Cleaning

* Handled missing values
* Verified data types
* Removed inconsistencies

### 2. Exploratory Data Analysis (EDA)

* Distribution analysis of key variables
* Correlation heatmap
* Feature relationships with mental_health_risk
* Identification of patterns and anomalies

### 3. Feature Analysis

* Selected numerical features for correlation analysis
* Filtered weak correlations to improve visualization clarity
* Built a **Top 10 Predictors chart**

### 4. Visualization

* Heatmaps for correlation analysis
* Bar charts for feature importance
* Distribution plots for key variables

---

## 📈 Key Insights

* **Stress level and depression score** are among the strongest predictors of mental health risk
* **Sleep and physical activity** show meaningful relationships with mental health outcomes
* **Substance use and prior mental health history** are associated with increased risk
* Weak correlations were removed to improve clarity and focus on meaningful relationships
